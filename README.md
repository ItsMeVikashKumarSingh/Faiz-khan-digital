# Faiz Khan Digital

![Faiz Khan Digital Banner](public/banner.png)

## Overview

**Faiz Khan Digital** is a premium, high-performance digital marketing agency website built with modern web technologies. Designed to convert visitors into clients, it features a stunning "Cyber-Elegant" aesthetic with glassmorphism effects, smooth animations, and a fully responsive layout.

## üöÄ Tech Stack

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Language:** TypeScript

## ‚ú® Key Features

- **Hero Section:** Immersive animated background with gradient orbs and 3D-like effects.
- **Smart Header:** Auto-hides on scroll down and reappears on scroll up for better UX.
- **Service Showcases:** Interactive grid layouts for Services, Packages, and Courses.
- **Real-Time Stats:** Animated counters for social proof.
- **Video Testimonials:** integrated video player modal for client success stories.
- **Responsive Design:** Optimized for all devices, from mobile phones to large desktops.
- **Performance:** Static site generation (SSG) for lightning-fast load times.

## üõ†Ô∏è Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/faiz-khan-digital.git
    cd faiz-khan-digital
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Open in browser:**
    Navigate to [http://localhost:3000](http://localhost:3000).
